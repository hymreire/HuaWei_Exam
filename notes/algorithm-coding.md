# 算法代码题

## 高频题型

### 动态规划

- 一维 DP
- 二维 DP
- 背包问题
- 最长递增子序列
- 最长公共子序列
- 编辑距离

### 贪心

- 区间调度
- 排序后选择
- 局部最优推出全局最优
- 反例检查

### 搜索

- DFS
- BFS
- 回溯
- 图连通性

## 常见套路

- 先定义状态，再写转移
- 先排序，再贪心
- 先画小样例，再找递推关系
- 输入规模决定复杂度
